
# GATracker (PxPlugin)


## インストール方法

1. Pickles Framework をセットアップする。
2. ディレクトリ plugins/GATracker を、Pickles Framework の plugins ディレクトリにアップロードする。


## 使い方

1. ブラウザで、PxCommand "?PX=plugins.GATracker" にアクセスする。


## システム要件

- Pickles Framework 1.0.0 以上
